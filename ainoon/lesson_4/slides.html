What is Vibe-coding?

TODO

First, what is coding?

Coding how software developers build apps - writing instructions in a special language the computer can understand.

Open the source view (Ctrl-u) on the technoon.org website to show the code for that website.

Vibe-coding is describing an app to an agent and letting it write the code without checking the code it writes - https://simonwillison.net/2025/Mar/19/vibe-coding/
